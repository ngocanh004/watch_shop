# Client - Watch Shop (Vue 3 + Vite + Pinia)

Website bán hàng phía khách hàng.

## Cài đặt

```bash
copy .env.example .env
npm install
npm run dev
```

Mặc định chạy tại `http://localhost:5173`.

Yêu cầu backend đã chạy tại `http://localhost:8080`.

## Tài khoản dùng thử

Username: `user01` / Password: `User@123`

## Tính năng

- Trang chủ với banner, sản phẩm nổi bật, sản phẩm mới, danh mục, thương hiệu
- Danh sách sản phẩm với bộ lọc nâng cao (danh mục, thương hiệu, giới tính, loại máy, loại dây, khoảng giá)
- Chi tiết sản phẩm với thông tin đầy đủ, tab mô tả/thông số
- Giỏ hàng (thêm/sửa/xoá/xoá tất cả)
- Danh sách yêu thích
- Đặt hàng (Checkout) với thông tin giao hàng
- Lịch sử đơn hàng theo trạng thái, chi tiết, huỷ đơn
- Đăng ký, đăng nhập, đổi mật khẩu, cập nhật cá nhân
- Responsive cho thiết bị di động
