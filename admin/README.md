# Admin - Watch Shop (Vue 3 + Vite + Pinia)

Trang quản trị cho website bán đồng hồ.

## Cài đặt

```bash
copy .env.example .env
npm install
npm run dev
```

Mặc định chạy tại `http://localhost:5174`.

Yêu cầu backend đã chạy tại `http://localhost:8080`.

## Tài khoản đăng nhập

Username: `admin01` / Password: `Admin@123`

## Tính năng

- Dashboard thống kê tổng quan
- Quản lý sản phẩm (CRUD, upload ảnh, lọc)
- Quản lý danh mục, thương hiệu
- Quản lý đơn hàng, cập nhật trạng thái
- Quản lý người dùng, reset mật khẩu
- Đổi mật khẩu, cập nhật thông tin cá nhân
